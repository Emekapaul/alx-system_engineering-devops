pwd prints the absolute path name of the current working directory

ls is used to Display the contents list of your current directory.

ls -l is used Display current directory contents in a long format.

ls -al is used to Display current directory contents, including hidden files i long format.

ls -lida is used to Display current directory long format with user id and hidden files.
